# Cho Thuê Xe Tự Lái Limo Green Hà Nội

Giao diện landing page chuyên nghiệp cho dịch vụ cho thuê xe điện 7 chỗ Limo Green tại Hà Nội. Được tối ưu hóa cho SEO 2026 và tích hợp sẵn công nghệ GEO (Generative Engine Optimization).

## Tính năng nổi bật
- **Giao diện cao cấp**: Sử dụng Tailwind CSS, AOS Animation và Swiper Slider.
- **Tối ưu chuyển đổi**: Tích hợp nút Zalo nổi, Hotline và Form đăng ký tự động chuyển hướng Zalo.
- **Chuẩn SEO/GEO**: Đã tích hợp Schema JSON-LD (LocalBusiness, Service) và bộ thẻ meta 2026.
- **Thư viện ảnh nội bộ**: Toàn bộ hình ảnh được lưu trữ tại `assets/images/`, không phụ thuộc link ngoài.

## Cách triển khai lên GitHub Pages
1. Tạo một repository mới trên GitHub.
2. Upload toàn bộ các file trong thư mục này (bao gồm `index.html`, `assets/`, `README.md`, `.gitignore`).
3. Vào phần **Settings** -> **Pages**.
4. Tại mục **Branch**, chọn `main` và ấn **Save**.
5. Đợi 1-2 phút, bạn sẽ có link website hoạt động 24/7 hoàn toàn miễn phí.
